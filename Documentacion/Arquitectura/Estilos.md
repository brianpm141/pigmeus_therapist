# Sistema de Estilos y Tema

El proyecto utiliza **NativeWind**, permitiendo el uso de clases de utilidad de Tailwind CSS. La configuración se define en `tailwind.config.js`.

## Tipografía
Se utiliza la fuente **Manrope** en diferentes pesos:
- `sans`: Manrope Regular (400)
- `medium`: Manrope Medium (500)
- `bold`: Manrope Bold (700)
- `extrabold`: Manrope ExtraBold (800)

## Paleta de Colores

### Principal (Marca)
- **Primary:** `#13c8ec` (Acciones principales)
- **Dark:** `#0ea5c3` (Estados hover/active)
- **Foreground:** `#FFFFFF` (Texto sobre primario)

### Fondos (Backgrounds)
- **Light:** `#f6f8f8`
- **Dark:** `#101f22`

### Superficies (Tarjetas/Contenedores)
- **Light:** `#ffffff`
- **Dark:** `#192b2e`
- **Darker:** `#142225`

### Bordes
- **Light:** `#e2e8f0`
- **Dark:** `#334155`

### Estados
- **Success:** `#22c55e`
- **Warning:** `#fb923c`
- **Danger:** `#ef4444`
- **New:** `#a855f7`

### Texto
- **Primary:** `#0f172a` (Modo claro)
- **Secondary:** `#94a3b8` (Subtítulos/Hints)
- **Inverse:** `#e2e8f0` (Modo oscuro)

## Bordes Redondeados
- **Default:** `4px`
- **lg:** `8px`
- **xl:** `12px` (Estándar para inputs)
- **2xl:** `16px`
- **3xl:** `24px`
